Memory Game

Memory Game est un jeu de mémoire en ligne développé avec Next.js, proposant une interface simple et ludique. Le principe est classique : 16 cartes retournées, dont il faut retrouver les 8 paires en cliquant pour révéler les facettes. Lorsqu’une paire est trouvée, elle reste visible ; sinon, les cartes se retournent à nouveau.


Ce projet met l’accent sur l’interactivité et la logique de jeu, avec une gestion fine de l’état des composants et du rendu conditionnel dans React.


Memory Game a été réalisé dans le cadre de la formation à La Capsule, et m’a permis de consolider mes bases en React/Next.js, en travaillant sur la gestion d’état, les animations simples, et l’expérience utilisateur dans un contexte de jeu.
