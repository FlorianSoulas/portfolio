Morning News

Site journalistique

Morning News est un site web d’actualité développé avec Next.js pour le frontend, Express pour le backend, et une base de données MongoDB. Il utilise l’API NewsAPI pour récupérer et afficher les actualités du jour en temps réel.


L’objectif de ce projet est de proposer une interface claire et efficace pour consulter les principales informations quotidiennes. Les utilisateurs peuvent parcourir les titres du jour, découvrir les articles en détail, et se tenir informés rapidement.


Ce projet a été réalisé en une journée dans le cadre du programme de formation à La Capsule. Il m’a permis de mettre en pratique les bases du développement web fullstack, l’intégration d’API tierces, ainsi que la structuration d’un backend simple pour orchestrer les échanges de données.
