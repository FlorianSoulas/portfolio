Weather App

Application web métérologique

Weather App est une application web développée avec HTML, CSS et JavaScript pour le frontend, et Node.js avec Express pour le backend. Elle s’appuie sur l’API OpenWeatherMap pour récupérer les données météo en temps réel.


L’objectif de cette application est de permettre aux utilisateurs de suivre facilement la météo de leurs villes préférées. L’interface simple et épurée affiche les conditions météorologiques actuelles, rendant l’expérience rapide et accessible.


Ce projet a été réalisé en une journée dans le cadre du programme de formation à La Capsule. Il m’a permis de mettre en pratique les bases du développement web fullstack, l’intégration d’API tierces, ainsi que la structuration d’un backend simple pour orchestrer les échanges de données.
